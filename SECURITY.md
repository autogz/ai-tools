# Security Policy

## Supported Versions

Only the latest PyPI release is supported.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities. Email the
maintainer (see GitHub profile for contact). Acknowledgment within 72 hours.

## Scope

- In scope: code execution, command injection, data leakage, unsafe output.
- Out of scope: third-party API vulnerabilities (OpenAI, Etherscan).

## Safe Harbor

Good-faith researchers reporting vulnerabilities will not face legal action.
